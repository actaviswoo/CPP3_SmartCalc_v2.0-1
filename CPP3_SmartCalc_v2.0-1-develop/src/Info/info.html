smartcalcV2 by argoniaz

About the program

Scientific calculator with the ability to calculate the expression f(x) with plotting. It is possible to calculate a credit or deposit

Installation

To install the calculator, type make install

Application launch

After installing the application, you can run the application through the terminal by running make start or manually opening smartcalcV2.app on the desktop in directory Argoniaz

Removing apps

To remove the calculator, write the command make uninstall
